# Devedoras_B3
Análise das empresas com maiores dívidas na B3
Este projeto tem como objetivo realizar uma análise fundamentalista e técnica das ações devedoras na B3, visando analisar o retorno dessas ações. Para alcançar esse objetivo, foi utilizado o Python como linguagem de programação, além de técnicas de limpeza de dados e criação de gráficos.
